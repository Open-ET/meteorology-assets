# CONUS404 Ancillary Assets

## UCAR

The "conus404_ancillary_ucar.py" script can be used to generate the ancillary assets needed to compute reference ET and mask ocean pixels.  This script will read the constant WRF fields from the invariant folder on the UCAR thredds server.

You will need to add a config.py file to this folder with "ucar_username" and "ucar_password" variables.
